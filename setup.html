Bot Setup Guide

Welcome to the Bot Setup Guide. Follow these steps to set up and run your bot.

Step 1: Install Python

Ensure you have Python 3.11.9 installed. If not, download and install it from the Python website.

Step 2: Install Dependencies

Open a command prompt and run the following commands to install the required dependencies:

pip install requests twitchio

Step 3: Navigate to Bot Directory

Open File Explorer, navigate to the directory where your bot script is located. Click on the address bar where the path is displayed (e.g., C:\path\to\your\bot\directory), type cmd, and press Enter. This will open a command prompt in the correct directory.

Step 4: Run the Bot

Run the bot script by executing the following command in your command prompt:

python index.py

Bot Configuration

The bot will prompt you to enter the necessary credentials and settings when you run it for the first time. You will be asked for:

  • Twitch Client ID
  • Twitch Client Secret
  • Twitch OAuth Token
  • Twitch Refresh Token
  • Bluesky Handle
  • Bluesky Password
  • Initial Twitch Channels
  • Whitelisted Users

The configuration will be saved in a config.json file for future use.

Troubleshooting

If you encounter any issues, make sure to check the following:

  • Python and pip are correctly installed.
  • All dependencies are installed.
  • Check the terminal output for any error messages and follow the instructions provided.
